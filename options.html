Tip: create an internal Notion integration and share the database with it.

You can paste the full database URL (Copy link) or the database ID.

Important

  • Les noms de colonnes Notion doivent correspondre exactement: Application Date, Date d'ouverture, Date de fermeture, Entreprise, Job Title, lien offre, Lieu, Role, Start month, Status, Type d'infrastructure.
  • Si tu as d'autres noms, modifie-les dans background.js (fonction buildProps).
